### INSTALATION

{" "}

<div class="not-prose text-red-700">
  **[VERY IMPORTANT] Must read before you even consider installing EvoWeb
  patch**
</div>

Installing the patch on a clean game is the only reliable way of installing EvoWeb patch
DO NOT ASSUME that EvoWeb patch is compatible with any of the mods you already have installed - be it through .cpk files, or through sider
Keep in mind that most of the existing mods are made using Konami's clean game as a baseline, to try to fix various shortcomings of the clean, unmodded game - big patches, such as EvoWeb patch (and many other patches as well), include their own fixes for the same shortcomings that mini-mods also cover.
Therefore, IT IS NOT POSSIBLE to use both a big patch (such as EvoWeb patch) and all of your old mini-mods simultaneously without having serious conflicts, since both of them try to do the same thing in a different way.
IF YOU KEEP USING THEM simultaneously, expect many issues:

wrong rosters
wrong team logos/names
missing teams
missing players and/or transfers
etc.

What to do then, to avoid conflicts?

Disable all mods you are using BEFORE installing EvoWeb patch
remove every custom .cpk from dpfilelist (if you used dpfilelist/.cpk's for modding)
disable every custom cpk.root folder and/or .lua script in sider (if you modded your game using sider)
Install EvoWeb patch according to the instructions you'll find later in this post
After installing, test the game to see if patch is installed correctly

But I still want to use some of the older mods, because EvoWeb patch does not include option XYZ!

Try reactivating old mods one by one, after you've ensured that EvoWeb patch is installed correctly - and always check if re-enabled mod causes any issues in game.
If Mod X gives you any issue with EvoWeb patch, consider that mod incompatible and disable it again. Either wait for an updated/new version, which will be compatible with EvoWeb patch or stop using that mod.

TL/DR: Do not complain about your post-install errors or issues if you haven't followed above-mentioned advices! Read it BEFORE YOU INSTALL EvoWeb patch and follow the suggestions.

How to install?

1. Install the newest Version of Sider (get familiar with it in case you never used it)
2. Important to already have DLC 7 installed via Steam and started the game once before installing this Version
3. If you installed a previous EvoWebPatch Version, remove the entire EvoWeb Patch folder from your livecpk folder first. Same with other Patches you may have installed, remove them fully
4. Add the livecpk Folder from the Download to your Sider folder
5. Open sider.ini and add the following cpk roots:

   <div class="not-prose bg-[#282c34] text-[#abb2bf] p-4 rounded-md">
     cpk.root = ".\livecpk\EvoWeb Patch\Boots-Gloves-AddOn" <br />
     cpk.root = ".\livecpk\EvoWeb Patch\Boots-Gloves" <br />
     cpk.root = ".\livecpk\EvoWeb Patch\Database" <br />
     cpk.root = ".\livecpk\EvoWeb Patch\Faces" <br />
     cpk.root =".\livecpk\EvoWeb Patch\Graphics" <br />
     cpk.root = ".\livecpk\EvoWeb Patch\Kits" <br />
   </div>

6. Add the EDIT00000000 file to Documents\KONAMI\eFootball PES 2021 SEASON UPDATE\Your number\save
7. Clean your WEPES & mount folder if you added any Option Files before
8. Start Sider and play the game
