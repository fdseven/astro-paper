### How to Install

1. Choose your preferred version (v1-v10).
2. Copy 3D Turf folder to your livecpk folder.
3. Add the following line to your sider.ini: <br />
   `cpk.root = ".\livecpk\3D Turf"` <br />
   In case you use another turf mod, make sure that the 3D Turf line is above the turf mod's line in your sider.ini.

   **Example:** <br />
   `cpk.root = ".\livecpk\3D Turf"` <br />
   `cpk.root = ".\livecpk\real turf"`

If you use Stadium Server, install the mod following the instructions above, then go into stadium-server folder, open the search bar (press CTRL + F) and search for **grassfin_bsm_alp.ftex**. Select all the files found and delete them. (press CTRL + A then DEL) Now the game will load the 3D Turf from livecpk both for Konami and Stadium Server stadiums.
