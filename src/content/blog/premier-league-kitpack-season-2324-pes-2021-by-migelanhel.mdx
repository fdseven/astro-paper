## Here's English Premier League Kitpack by Migelanhel

<hr />

### Features

- All Premier League 23/24 kits updated
- Completed: Premier League, Emirates FA Cup, The FA Community Shield, UEFA Champions League, UEFA Europe League, UEFA Super Cup
- New Season 23/24 EPL Database and Transfers squad
- New Season 23/24 EPL Champions + All teams badge
- New Season 23/24 La-Liga hypermotion logo
- New Season 23/24 UEFA Champions League İntro 1080p [60fps]

### How to Install

- Install Version 7.1.4 of Sider
- You need Sider 7.1.4 as the new used kit values only work with this Release
- Uninstall existing Premier League kits package for a clean install !
- Extract "\_Migelanhel_Edits" file from rar.
- Throw file into sider and say yes to all the questions.
- Add/replace the EDIT00000000 file to Documents\KONAMI\eFootball PES 2021 SEASON UPDATE\Your number\save
