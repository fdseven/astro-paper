### General notices:

- Please read the FAQ's as it may already answers your question
- The Database of this Patch was built with bin files, it's important you don't use other Mods using Database bin files like Manager Mods.
  You need to disable them and ask the Creator for an Update. ML Manager Mod by @SoulBallZ needs an Update, otherwise the Patch won't work. Get it HERE
- The Status of the Database is the eFootball Database from March 2024 (including Winter Transfers) with manual adjustments to League/National Teams
- It's highly suggested to use the Commentary/Chants Mod by @predator002 with this Patch
  Please note that you don't use the EvoWeb Patch chantlists anymore he provided as they are not needed anymore because of the new real IDs

### Information about League structures:

- Some Leagues previously have changed their amount of Teams. It's mostly not possible to change that in PES2021 because it results in crashes or wrong schedules
- Because of that reason we had to re-structure some Leagues to have a crash-free Master League experience
- Ligue 2 has been changed from 20 to 22 Teams (Dijon & Nimes added with 23/24 squads). This way an 18 Team Ligue 1 is possible
- Suisse Super League has 10 instead of 12 Teams (Lausanne-Sport & Stade Lausanne Ouchy has been moved to Other European)
- Liga Argentina has 26 instead of 28 Teams (Arsenal & Colon has been moved to Other Latin America Teams)
- Jupiler Pro League has 18 instead of 16 Teams (KV Oostende & Zulte-Waregem added with 23/24 squads)

### INSTALATION

{" "}
<div class="not-prose text-red-700">
  **[VERY IMPORTANT] Must read before you even consider installing EvoWeb
  patch**
</div>

1. Installing the patch on a clean game is the only reliable way of installing EvoWeb patch
2. DO NOT ASSUME that EvoWeb patch is compatible with any of the mods you already have installed - be it through .cpk files, or through sider
3. Keep in mind that most of the existing mods are made using Konami's clean game as a baseline, to try to fix various shortcomings of the clean, unmodded game - big patches, such as EvoWeb patch (and many other patches as well), include their own fixes for the same shortcomings that mini-mods also cover.

**Therefore, IT IS NOT POSSIBLE to use both a big patch (such as EvoWeb patch) and all of your old mini-mods simultaneously without having serious conflicts, since both of them try to do the same thing in a different way.
IF YOU KEEP USING THEM simultaneously, expect many issues:**

- wrong rosters
- wrong team logos/names
- missing teams
- missing players and/or transfers
- etc.

### What to do then, to avoid conflicts?

1. Disable all mods you are using BEFORE installing EvoWeb patch


    - remove every custom .cpk from dpfilelist (if you used dpfilelist/.cpk's for modding)
    - disable every custom cpk.root folder and/or .lua script in sider (if you modded your game using sider)

2. Install EvoWeb patch according to the instructions you'll find later in this post
3. After installing, test the game to see if patch is installed correctly

**But I still want to use some of the older mods, because EvoWeb patch does not include option XYZ!**

1. Try reactivating old mods one by one, after you've ensured that EvoWeb patch is installed correctly - and always check if re-enabled mod causes any issues in game.
2. If Mod X gives you any issue with EvoWeb patch, consider that mod incompatible and disable it again. Either wait for an updated/new version, which will be compatible with EvoWeb patch or stop using that mod.

**TL/DR: Do not complain about your post-install errors or issues if you haven't followed above-mentioned advices! Read it BEFORE YOU INSTALL EvoWeb patch and follow the suggestions.**

### How to install?

1. Install the newest Version of Sider (get familiar with it in case you never used it)
2. Important to already have DLC 7 installed via Steam and started the game once before installing this Version
3. If you installed a previous EvoWebPatch Version, remove the entire EvoWeb Patch folder from your livecpk folder first. Same with other Patches you may have installed, remove them fully
4. Add the livecpk Folder from the Download to your Sider folder
5. Open sider.ini and add the following cpk roots:


    ```ini
    // sider.ini
    cpk.root = ".\livecpk\EvoWeb Patch\Boots-Gloves-AddOn"
    cpk.root = ".\livecpk\EvoWeb Patch\Boots-Gloves"
    cpk.root = ".\livecpk\EvoWeb Patch\Database"
    cpk.root = ".\livecpk\EvoWeb Patch\Faces"
    cpk.root =".\livecpk\EvoWeb Patch\Graphics"
    cpk.root = ".\livecpk\EvoWeb Patch\Kits"
    ```

6. For the Euro 2024 Minifaces Server add the following lua module line under CommonLib.lua


    ```ini
    // sider.ini
    lua.module = "MiniFaceServer.lua"
    ```

7. Add the content & modules folder from the Euro 2024 Minifaces Server folder into your Sider folder and replaces existing files if asked
8. Add the EDIT00000000 file to Documents\KONAMI\eFootball PES 2021 SEASON UPDATE\Your number\save
9. Clean your WEPES & mount folder if you added any Option Files before
10. If you added other Database related files before, like ML Manager Mod, make sure it is compatible with the Patch Version you are using
11. Start Sider and play the game
