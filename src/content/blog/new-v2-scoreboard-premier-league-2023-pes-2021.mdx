## Proudly present & release my version 2 of scoreboard of Premier League 2023 for PES 2021

<hr />

### Some graphic improvements from [version 1](https://www.pesmania.web.id/posts/the-new-scoreboard-premier-league-2023/) :

- Colors
- New pause menu
- Statistic / result team record
- Aston Villa old emblem
- New tv channels :
  - Sky Sports
  - DAZN
  - Directv
  - CBS Sports
  - And also no tv channel logo & no Premier League 2023/24 watermark

Optional : <br />
I also update the luma margin value to default (165) for contrast team name color at modules/ScoreboardServer.lua. And update the kit unicolor of Manchester City from https://teamcolorcodes.com/manchester-city-fc-colors/
So Machester City will be the light team name color (white color).

CODE : <br />
`local luma_margin = 165`

And then update config file in Man. City kit folder --> P1/P1-EPL (as you have) in kit server folder

CODE : <br />
`UniColor_Color1= #6CABDD`

**Add this code to map_cometitions.txt (depends on your scoreboard-server folder structure) :** <br />
17, England\EPL\PL23-V2-vidio <br />
17, England\EPL\PL23-V2-bein <br />
17, England\EPL\PL23-V2-skysports <br />
