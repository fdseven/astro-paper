### How to Install PES 2021 Faces:

Installing the New PES 2021 Facepack& is a simple process that can be done in a few steps. Here's a quick guide to help you get started:

#### Sider Version

Extract and copy the folder with ID (numbers) along the path: <br />
`... \ livecpk\Faces\Asset\model \character\face\real`

Open sider.ini, and add the following line (if not already done)

{" "}

{" "}
<pre>
  <code>
    <span class="text-white">cpk.root = ".\livecpk\faces"</span>
  </code>
</pre>

#### CPK Version

Use "CriPackedFileMaker" to create a "CPK" file of "Sider version" folder contents. <br />
Copy "CPK" file to eFootball PES2021\download

Use the "DpFileList Generator" tool to add the created CPK file into your game
