  ## PES 2021 Corrected Sun Direction - Project Helios made by Dragovic - This mod corrects the sun position of stadiums with inaccurate sun position

### Features :

1. Corrected the sun position of the base light files
2. Added the same sun position settings in fog files and corrected some shadow intensity (if there is any fog file for that stadium)
3. Increased sunlux values for some stadiums

**Disclaimer:**
All of the light files are the property of the stadium creators. Furthermore, this mod is not intended for commercial use by any paid patches.
The mod is only for manipulating day lighting. Please don't come up to me with issues in Night mode. As for the above pics, these were taken in ML during different in-game times, so the exhibition mode lighting may differ due to different times, but it will work.

### Instructions to install:

1. Download the respective stadium files from the link below
2. Copy or overwrite the files in the respective stadium folders to your Sider folder
3. If there is an Addon folder, I recommend configuring your addon_config.txt file

**NOTE: If my files don't contain any Addon, then check if your Addon folder (if you have one) has any light files and deactivate.**
