### How to Install

1. Download & install Ball Server ..Instructions [HERE](https://www.pesmania.web.id/posts/ball-server-pes-2021-game-experience-exciting)
2. If not already done, download & install CommonLib module [HERE](https://modsfire.com/qLU7isnb42UsXDC)
3. Once BallServer & CommonLib are installed copy the "ball-server" folder from the BallServer Pack 42 AIO.RAR archive to your Sider "content" folder. Delete any other ball packs first..this is an AIO pack ...See NOTE 2 below.

**NOTE**: <br />

- Don't try to mix ball packs or you will face problems/errors....you can of course add your own balls after installing this pack.
- If the patch your using came with a ballserver pack installed you must delete it before installing this pack. Failure to do so WILL result in errors that I can't fix.
