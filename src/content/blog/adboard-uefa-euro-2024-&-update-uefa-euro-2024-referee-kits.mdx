## From Nguyen Xuan Duc

### Instructions

**Step 1:** <br/>
Extract the RAR file

**Step 2:** <br/>
Copy the Asset and common folders to the Adboars folder in your Sider

**Step 3:** <br/>
Go to the following path in your Adboards folder `common\bg\model\bg\bill\config`, open the config file and add this code and save

```
<condition stid="-1" compeid="33" hometeamid="-1" teamexclusion="-1" language="-1" region="-1" title="-1" pattern="uefaeuro2024" />
```

**Step 4:** <br/>
Open the stadium_server folder in the sider's content folder. Here I used Marlon Anthony UEFA Euro 2024 stadium set as the main base. Copy the common folder in the stadium folders that I placed in the download folder into the stadium folders that you linked to the Euro 2024 tournament in the game.

**Step 5:** <br/>
Enter the game and enjoy

It is possible that some stadiums will display advertising signs with high contrast, but this is due to the stadium lighting and I cannot intervene.
Note: Your stadiums need to be linked to UEFA Euro 2024 via Stadium_Server for the most accurate display
