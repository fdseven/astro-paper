So for all who want it a bit easier you can start with this AddOn!
It delivers kit-server extracted from SPFL24 using kitstudio - new balls - new kits and fonts LaLiga 1+2, EFL and GSL Season 23/34.

**Tips on how to use addons at SP Footbal Life 24**

1.  Any database related mods which use player.bin, team.bin or any other pesdb files (manager, competition mods) is very hard to implement, SP FL24 has crypted the player db file and some other, so structure is unknown (Based on Konami IDs).
2.  Use sider - easiest way to activate addons or update SPFL24 preinstalled content
3.  Read the sider manual: https://mapote.com/doc/sider/
4.  Some knowledge about how to edit textfiles like lua modules with id structure
5.  All grafix mods can be used, but you must edit any player or team id related content to FL24 id's (like lua modules or map.txt's, player faces and so on).
6.  Database ID's: https://www.pessmokepatch.com/2022/07/fl23teams.html
7.  Look at the files in modules, content, livecpk and sider.ini - if not sure make a copy of the unmodded folder before modding
8.  That's all - read, understand, mod :)

Install.txt is delivered, just in case:

### INSTRUCTIONS

1. Copy folder content, livecpk and modules into FL24/SiderAddons
2. Open sider.ini in FL24/SiderAddons and overwrite the line <br />
   **'lua.module = "common\kits.lua"'** with this: <br />
   `;lua.module = "common\kits.lua"` <br />
   `lua.module = "common\kserv.lua"` <br />

   and add this line: <br />
   `cpk.root = ".\livecpk\Fonts"`

3. Kit-server is now installed. You can now select kits ingame via kitserver by using sider overlay.
   How to use:
   https://mapote.com/doc/sider/sider7/readme.html
4. If you want to add any kits, go to FL24/SiderAddons/content/kit-server and search for the team or open map.txt and search there.
5. Copy sider related kits into the corresponding folder and make sure to update gk_order.ini and order.ini
6. If you want some new balls extract omb.zip to FL24/SiderAddons/content and overwrite existing files.
