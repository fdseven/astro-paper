### How to install?

1. Install Evo Web Patch 2024 Version 1.0 if you haven't already.
2. Go to your livecpk/EvoWeb Patch folder and delete the folder "Kits". This step is very important, copy and replace will result in issues.
3. If you use Kitserver configs go to your content folder and delete the entire "kit-server" folder.
4. Now add the EvoWeb Patch folder from the Download to your livecpk folder.
5. If you use Kitserver configs add the kit-server folder from the Download to your content folder.
6. I highly suggest you to delete your EDIT00000000 file from Documents\KONAMI\eFootball PES 2021 SEASON UPDATE\Your number\save and generate
   a new one ingame if you haven't done major Edits on your Editfile because you will possibly get blank kits on kit selection if you don't.
7. Start Sider and play the game
